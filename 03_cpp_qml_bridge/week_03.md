# Неделя 03: Связка C++ и QML

## Цели

- создать backend-объект на C++
- передать свойства в QML
- вызывать методы C++ из QML
- отправлять обновления из C++ в интерфейс

## Основные темы

- `QObject`
- `Q_PROPERTY`
- `Q_INVOKABLE`
- `signals`
- регистрация типов

## Мини-проект

Сделать небольшую панель состояния устройства, где QML показывает данные из C++.
