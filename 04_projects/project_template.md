# Шаблон учебного проекта

## Цель проекта

Собрать небольшое desktop-приложение на Qt 6 + QML, где UI отделен от backend и есть понятная структура файлов.

## Рекомендуемая структура

- `src/`
- `src/main.cpp`
- `src/backend/`
- `src/models/`
- `qml/`
- `qml/pages/`
- `qml/components/`
- `assets/`
- `CMakeLists.txt`

## Минимальный состав

- одно главное окно
- две страницы
- один backend-объект на C++
- один список данных
- один переиспользуемый компонент

## Что считаем хорошим результатом

- проект собирается без ручной магии
- имена файлов и компонентов понятны
- логика не размазана случайно между QML и C++
- интерфейс уже пригоден для дальнейшего расширения
