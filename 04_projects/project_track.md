# Трек учебных проектов

## Рекомендуемые проекты

1. Панель управления устройством
2. Приложение для заметок
3. Менеджер задач
4. Утилита настроек

## Лучший первый проект

Панель управления устройством особенно хорошо подходит, если ты идешь в Qt из embedded C и работы с железом.

В таком проекте стоит сделать:

- карточки состояния
- страницу настроек
- список данных
- баннер ошибок
- mock-backend на C++
