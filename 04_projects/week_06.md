# Неделя 06: Сетевые данные, JSON и файлы

## Цели

- научиться получать и обрабатывать внешние данные
- разобраться, что лучше делать в QML, а что в C++
- начать думать как разработчик прикладного desktop-приложения

## Темы

- `QNetworkAccessManager`
- JSON
- чтение и запись файлов
- отображение ошибок загрузки

## Практика

- загрузить тестовые данные
- распарсить JSON в C++
- показать результат в интерфейсе
- отобразить ошибку, если данные не загрузились

## Результат недели

У тебя появляется приложение, которое работает не только со статическим UI, но и с реальными данными.
