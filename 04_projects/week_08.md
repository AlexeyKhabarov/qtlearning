# Неделя 08: Завершение учебного проекта

## Цели

- довести один проект до внятного законченного состояния
- провести ревизию архитектуры
- закрепить ключевые темы курса

## Темы

- рефакторинг
- производительность QML на базовом уровне
- `qmllint`
- `qmlformat`
- подготовка к упаковке приложения

## Практика

- выбрать один учебный проект как основной
- убрать лишнюю связанность между UI и backend
- пройтись по naming и структуре файлов
- составить список того, что нужно изучать дальше

## Итог

К концу недели у тебя есть небольшой, но настоящий проект, который не стыдно показывать как учебный результат.
