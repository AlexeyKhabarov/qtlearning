# Словарь

- Qt: кроссплатформенный C++ фреймворк для разработки приложений
- Qt Quick: технология построения современных интерфейсов в Qt
- QML: декларативный язык для описания пользовательского интерфейса
- Binding: автоматическое обновление свойства на основе другого свойства
- Component: переиспользуемый строительный блок интерфейса в QML
- `QObject`: базовый класс, на котором построены многие объектные возможности Qt
