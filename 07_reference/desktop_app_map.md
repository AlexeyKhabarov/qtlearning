# Карта современного desktop-приложения на Qt

## Что обычно лежит в QML

- структура экранов
- визуальные компоненты
- простое состояние интерфейса
- анимации
- обработка пользовательских действий

## Что обычно лежит в C++

- бизнес-логика
- работа с файлами
- сеть
- модели данных
- интеграция с устройствами
- сервисы приложения

## Главное правило

Если логика может понадобиться вне одного экрана или быстро разрастается, скорее всего ей место в C++, а не в QML.
